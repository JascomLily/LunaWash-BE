LunaWash là hệ thống quản lý rạp rửa xe tự động thông minh (Smart Car Wash Management System) tích hợp quản lý điểm thưởng (Loyalty Program) và đặt lịch theo hạng thành viên. Đây là repository chứa mã nguồn Backend (RESTful API).

## 🚀 Tính năng cốt lõi
* **Loyalty Engine:** Tích điểm, nâng/hạ cấp bậc thành viên (Member, Silver, Gold, Platinum).
* **Booking System:** Đặt lịch rửa xe ưu tiên theo hạng (Priority queue).
* **Admin Dashboard:** Cấu hình điểm thưởng, quản lý khuyến mãi và theo dõi lịch sử.

## 🛠 Yêu cầu hệ thống
* [.NET 8.0 SDK](https://dotnet.microsoft.com/download) (Hoặc version nhóm bạn dùng)
* SQL Server
